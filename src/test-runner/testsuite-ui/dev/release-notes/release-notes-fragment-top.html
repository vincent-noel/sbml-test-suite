The overall SBML Test Suite is the work of Sarah M. Keating, Lucian P. Smith, Michael Hucka, Frank T. Bergmann, Bruce E. Shapiro, Thomas W. Evans, Colin S. Gillespie, Darren J. Wilkinson, Brett Olivier, and Andrew Finney.

The SBML Test Suite lets you test the degree and correctness of SBML support provided in an SBML-compatible software package. It consists of (1) a collection of SBML models, each with associated simulation results, and (2) the SBML Test Runner, a GUI program to make applications run through each test case in the suite and report successes and failures. The SBML Test Runner is a Java program that uses the Eclipse SWT widget toolkit and can be used on macOS, Linux and Windows.

The test cases in the overall Test Suite are divided into 3 sets: semantic, stochastic, and syntactic. Each set is meant to exercise a different aspect of SBML understanding and compliance. The SBML Test Runner currently only implements facilities for running the semantic tests. The semantic portion of the Test Suite contains valid SBML models with known, deterministic simulation results. These models can be used to test the ability of a software system to understand the meaning of different SBML constructs and properly simulate the models in a deterministic fashion.
